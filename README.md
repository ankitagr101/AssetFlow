# AssetFlow

## About the Project

AssetFlow is a simple web application developed for the Odoo Hackathon. It helps an organization manage its assets and employees in one place. Users can add company assets, add employee details, assign assets to employees, and view the records through a simple dashboard.

## Features

- Login Page
- Dashboard
- Add and View Assets
- Add and View Employees
- Assign Assets to Employees
- SQLite Database
- Reports Page
- Simple and Clean Interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- SQLAlchemy

## Project Structure

- Dashboard
- Asset Management
- Employee Management
- Asset Assignment
- Reports

## How to Run

1. Install Python.
2. Install Flask and SQLAlchemy.
3. Run:

```bash
python app.py
```

4. Open:

```
http://127.0.0.1:5000
```
